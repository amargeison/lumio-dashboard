# Lumio Sports — Brand Universe

This document is the canonical reference for fictional entities used in
demo content across the Lumio Sports portals. Whenever demo content needs
to depict a relationship that doesn't exist in real life — a partnership,
a fixture, a transfer, a hospitality event, a sponsorship — the fabricated
entity in question MUST come from this document, never a real club / venue
/ governing-body relationship.

## Pattern 1 vs Pattern 2

Two patterns govern when to fabricate vs preserve:

- **Pattern 1 — REPLACE.** Real entities used inside *fabricated* relationships.
  Examples: "Saracens partnered with Hartfield RFC on...", "Old Trafford hosted
  our Lancashire fixture", "Bath Rugby loaned Jake Rawlings to us". These are
  imagined relationships and must use fictional clubs / grounds.

- **Pattern 2 — KEEP.** Real entities referenced in *factual / public-information*
  contexts. Examples: "RFU concussion protocol stipulates...", "ECB Hundred
  competition runs in August", "Edgbaston hosts T20 Blast Finals Day". These
  are statements of fact about the actual sport's structure and stay verbatim.

The Pattern 2 list of safe-to-keep real entities includes: RFU, Premiership
Rugby (as a competition name in regulatory context), Six Nations, Lions,
Gallagher Premiership (regulatory name), ECB, MCC (as an institution),
The Hundred, IPL, BBL, SA20, T20 Blast, County Championship, One Day Cup,
Edgbaston (in Finals Day context), FA, WSL, WSL2, FSR, Karen Carney Review,
PWR, FIFA, UEFA. When in doubt: factual = keep, fabricated = replace.

## How to apply

When editing demo content:

1. Decide if the reference is Pattern 1 (fabricated) or Pattern 2 (factual).
2. Pattern 2 → keep verbatim.
3. Pattern 1 → replace using a name from the appropriate table above.
4. Keep the demo coherent — if a fabricated fixture is "vs Saracens at
   Stade de la Garenne", it becomes "vs Northbridge Saxons at the Saxons'
   home ground" (no need to invent venue names per club; rugby clubs in
   demo content are referred to by club name only).
5. For cricket, pair counties with grounds when practical
   ("Calderbrook CCC at Westmoor Cricket Ground" reads better than
   inconsistent pairings).
6. For player names embedded in fabricated contexts — invent names matching
   the existing demo player roster style (anglo / culturally varied / first
   name + surname, no real-player echoes).
